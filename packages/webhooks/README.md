## Features

- Sync issue creation bi-directionally
- Sync issue closing bi-directionally
- Sync issue comments bi-directionally

## Development

1. Start IssueForge locally with `yarn start`.
2. On another terminal window, run `yarn start:expose` to link IssueForge with an actual endpoint to test the webhooks.
3. Edit files and have fun.
